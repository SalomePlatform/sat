# SAT -  SALOME Builder Tool

[![Run Tests](https://github.com/basile-simvia/sat/actions/workflows/test.yml/badge.svg)](https://github.com/basile-simvia/sat/actions/workflows/test.yml)


SAT is a tool developed in the Python language, that aims to provide a universal and unified approach for the preparation, compilation and packaging of the different components of the SALOME platform.

To configure and build SALOME with SAT, follow closely the recipe described at this [link](https://github.com/SalomePlatform/.github/wiki).
